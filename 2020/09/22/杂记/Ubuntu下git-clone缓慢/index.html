Ubuntu下git-clone缓慢

发表于 2020-09-22 分类于 杂记 阅读次数：

git clone缓慢的主要原因是国内DNS无法自动转跳到github.global.ssl.fastly.net，该域名被限制导致，通过在/etc/hosts加上对应的地址可以解决这个问题

  • 查询github.global.ssl.fastly.net对应域名的地址，在shell下输入

    查询到对应的ip如：151.101.77.194

  • 查询github.com对应的域名

    查询到对应的ip如：13.229.188.59
